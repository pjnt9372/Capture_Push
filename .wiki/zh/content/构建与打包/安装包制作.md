## 简介
Capture_Push 使用 **Inno Setup** 制作 Windows 安装包。Inno Setup 是一个免费且强大的安装程序制作工具，能够创建专业的 Windows 安装程序。本项目提供了两种安装包配置：完整版（包含 Python 环境）和轻量版（仅包含程序文件）。

## Inno Setup 脚本结构
安装包的核心定义在 `.iss` 文件中。主要包含以下部分：

- **[Setup]**: 定义应用程序名称、版本、发布者、默认安装目录、输出文件名等全局属性。
- **[Languages]**: 定义安装向导的语言（如简体中文）。
- **[Tasks]**: 定义用户可选的任务（如创建桌面快捷方式、开机自启）。
- **[Files]**: 定义需要打包的文件和目标路径。
- **[Icons]**: 定义开始菜单和桌面的快捷方式。
- **[Run]**: 定义安装完成后执行的操作（如初始化配置、启动程序）。
- **[UninstallDelete]**: 定义卸载时需要清理的额外文件。
- **[Code]**: Pascal 脚本代码，用于处理复杂的安装逻辑（如检测 Python 环境、版本对比）。

## 完整版与轻量版

### 完整版 (`Capture_Push_Setup.iss`)
- **目标用户**：普通用户，机器上可能没有 Python 环境。
- **特点**：
    - 内置嵌入式 Python 环境。
    - 体积较大。
    - 开箱即用，无外部依赖。
- **关键配置**：
    ```iss
    [Files]
    Source: "python\*"; DestDir: "{app}\python"; Flags: ignoreversion recursesubdirs createallsubdirs
    ```

### 轻量版 (`Capture_Push_Lite_Setup.iss`)
- **目标用户**：开发者或已安装 Python 的高级用户。
- **特点**：
    - 不包含 Python 环境。
    - 体积小巧。
    - 依赖系统路径中的 Python。
- **逻辑**：
    - 在 `[Code]` 段包含检测逻辑，检查注册表或环境变量中是否存在 Python。

## 本地化支持
项目包含 `ChineseSimplified.isl` 文件，用于将 Inno Setup 的标准界面翻译成简体中文，提供更好的用户体验。

## 打包流程
1.  **准备文件**：确保所有源码、资源文件、依赖库和编译好的托盘程序 (`Capture_Push_tray.exe`) 都已准备就绪。
2.  **编译脚本**：
    - 右键点击 `.iss` 文件，选择 "Compile"。
    - 或者使用命令行：`ISCC Capture_Push_Setup.iss`。
3.  **获取产物**：编译成功后，安装包将生成在 `Output` 目录中。

## 测试与验证
发布前应进行以下测试：
1.  **净室安装**：在干净的虚拟机中测试完整版安装，确保不依赖预装软件。
2.  **升级安装**：测试覆盖旧版本的安装流程，确保配置文件和数据保留。
3.  **卸载测试**：验证卸载程序能否干净地移除文件和注册表项。
4.  **功能验证**：安装后启动程序，检查托盘图标、配置界面和推送功能是否正常。

## 结论
通过 Inno Setup，Capture_Push 提供了专业且灵活的安装体验。维护者应定期检查 `.iss` 脚本，确保其与项目文件结构保持同步。
